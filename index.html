Projects

Showcased below are three of the projects I have worked on during the first trimester of the MITxPRO Coding program. The first is a PacMan Factory Game, the second a eye animation, and the third a live bus tracker project. If you hover over each project you will find a link (left) to the project in action and (right) to the GitHub repository for the project.

This project creates an interactive game with two buttons. The first button will add PacMen to the screen, and the second button will set the PacMen bouncing around the screen at random speeds.

This project creates an image of two eyes on the screen. The pupils of the two eyes will follow the movement of the cursor around the page as you move your mouse.

This project pulls up the map of the city of Cambridge and live-tracks busses along a route in the middle of the map. In order to view the functionality of this project, you will have to obtain your own API key through google and follow the directions in the README file in the repository.
